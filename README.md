심근경색 및 협심증 발병 확률 예측 웹 서비스

사용자의 건강검진 데이터를 기반으로 심혈관 질환 발병 위험도를 예측하고, 시각화를 통해 직관적으로 건강 상태를 관리할 수 있는 웹 서비스입니다.

## 개요
* **프로젝트 이름**: 심근경색 및 협심증 발병 확률 예측 웹 서비스
* **개발 기간**: 2026.05.13 ~ 2026.05.26
* **개발 언어**: Python, Java, JavaScript
* **팀원**: 강현기, 곽민재, 박성준, 김민성

## 웹 서비스 설명
본 프로젝트는 머신러닝 모델을 활용하여 사용자의 건강검진 데이터를 분석하고, 심근경색 및 협심증 발병 가능성을 확률로 산출합니다.

* **주요 기능**:
    * 건강검진 데이터 입력 인터페이스 제공
    * 머신러닝 기반 발병 위험도 실시간 예측
    * 데이터 시각화 대시보드를 통한 일별 위험도 변화 추적

---

## 웹 서비스 작동 방식

![메인 화면 사진](/Project2_1/이미지/메인화면.png)

웹 서비스는 크게 데이터 입력, 예측 처리, 결과 시각화의 3단계로 작동합니다.

1.  **데이터 입력**: 사용자가 자신의 건강검진 결과(혈압, 콜레스테롤, 혈당 등)를 입력하면, 프론트엔드(JavaScript)에서 입력값을 검증합니다.
2.  **머신러닝 예측 모델 구동**: 입력된 데이터는 백엔드(Flask 환경)로 전송됩니다. 사전에 학습된 머신러닝 모델이 데이터를 분석하여 심혈관 질환 발병 확률을 계산합니다.
3.  **데이터 시각화**: 산출된 확률 데이터는 데이터베이스에 저장되며, 사용자가 과거부터 현재까지 자신의 위험도 변화를 추세선 그래프 등으로 한눈에 확인할 수 있도록 시각화하여 제공합니다.
