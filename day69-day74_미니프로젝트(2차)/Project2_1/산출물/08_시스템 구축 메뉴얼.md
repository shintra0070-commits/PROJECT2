# 08 시스템 구축 메뉴얼

## 1. 문서 개요

본 문서는 AI 기반 심근경색 발생 확률 예측 서비스의 분석 및 서비스 개발환경 구축 절차를 정리한 시스템 구축 매뉴얼이다.  
모든 설치 및 실행 명령어는 Windows 명령 프롬프트(cmd.exe) 기준으로 작성한다.  
프로젝트는 React 프론트엔드, Spring Boot 백엔드, Flask AI 예측 서버, Oracle Database로 구성된다.

## 2. 시스템 구성

| 구분      | 기술        | 역할                                          | 기본 포트 |
| --------- | ----------- | --------------------------------------------- | --------- |
| Frontend  | React       | 사용자 화면, API 호출, 결과 시각화            | 3000      |
| Backend   | Spring Boot | 회원, 건강 데이터, 커뮤니티, 문의 REST API    | 8080      |
| AI Server | Flask       | 심근경색 예측 모델 로드 및 예측 API 제공      | 5000      |
| Database  | Oracle DB   | 회원/건강 데이터/예측 결과/게시판 데이터 저장 | 1521      |

## 4. 설치 순서

### 4.1 Git 설치 및 프로젝트 준비

1. Git을 설치한다.

```cmd
winget install --id Git.Git -e
```

2. 프로젝트 폴더로 이동한다.

```cmd
cd C:\Users\Administrator\Desktop\project\Project2_1
```

3. Git 설치 여부를 확인한다.

```cmd
git --version
```

### 4.2 Java JDK 17 설치

1. JDK 17을 설치한다.

```cmd
winget install --id Oracle.JDK.17 -e
```

2. Java 버전을 확인한다.

```cmd
java -version
```

정상 예시:

```text
java version "17.0.18"
```

3. 환경 변수 `JAVA_HOME`을 JDK 17 설치 경로로 설정한다.

```cmd
setx JAVA_HOME "C:\Program Files\Java\jdk-17"
setx PATH "%PATH%;%JAVA_HOME%\bin"
```

### 4.3 Oracle Database 준비

본 프로젝트는 Oracle Database 접속 정보를 사용한다.

| 항목     | 값                                         |
| -------- | ------------------------------------------ |
| DBMS     | Oracle Database                            |
| Host     | 210.119.12.109                             |
| Port     | 1521                                       |
| Service  | xe                                         |
| User     | project                                    |
| Password | 1234                                       |
| JDBC URL | `jdbc:oracle:thin:@210.119.12.109:1521/xe` |

Spring Boot 설정 파일 위치:

```text
backend/src/main/resources/application.properties
```

Flask DB 설정 파일 위치:

```text
flask_backend/db.py
flask_backend/config.py
```

DB 스크립트 실행 파일:

```text
DB산출물/06_User_Table_Data_스크립트.sql
```

Oracle SQL Developer 또는 SQL\*Plus에서 아래 순서로 실행한다.

```sql
-- 1. project 계정 접속
-- 2. 테이블 생성 및 초기 데이터 입력
@06_User_Table_Data_스크립트.sql
```

### 4.4 Spring Boot Backend 설치 및 실행

1. 백엔드 폴더로 이동한다.

```cmd
cd C:\Users\Administrator\Desktop\project\Project2_1\backend
```

2. Gradle Wrapper 버전을 확인한다.

```cmd
gradlew.bat --version
```

3. 의존성을 내려받고 빌드한다.

```cmd
gradlew.bat clean build
```

4. Spring Boot 서버를 실행한다.

```cmd
gradlew.bat bootRun
```

5. 실행 주소를 확인한다.

```text
http://localhost:8080
```

### 4.5 Python 및 Flask AI 서버 설치

1. Python을 설치한다.

```cmd
winget install --id Python.Python.3 -e
```

2. Python 버전을 확인한다.

```cmd
python --version
```

3. Flask 서버 폴더로 이동한다.

```cmd
cd C:\Users\Administrator\Desktop\project\Project2_1\flask_backend
```

4. 가상환경을 생성한다.

```cmd
python -m venv .venv
```

5. 가상환경을 활성화한다.

```cmd
.venv\Scripts\activate.bat
```

6. 필요한 패키지를 설치한다.

```cmd
python -m pip install --upgrade pip
python -m pip install flask flask-cors oracledb joblib numpy pandas scikit-learn
```

7. 모델 파일이 있는지 확인한다.

```text
flask_backend/models/heart_model_last.pkl
flask_backend/models/model_features.pkl
```

8. Flask 서버를 실행한다.

```cmd
python manage.py
```

9. 실행 주소를 확인한다.

```text
http://localhost:5000
```

### 4.6 Node.js 및 React Frontend 설치

1. Node.js를 설치한다.

```cmd
winget install --id OpenJS.NodeJS -e
```

2. Node.js와 npm 버전을 확인한다.

```cmd
node -v
npm -v
```

3. 프론트엔드 폴더로 이동한다.

```cmd
cd C:\Users\Administrator\Desktop\project\Project2_1\frontend
```

4. 패키지를 설치한다.

```cmd
npm install
```

5. React 개발 서버를 실행한다.

```cmd
npm start
```

6. 실행 주소를 확인한다.

```text
http://localhost:3000
```

## 5. 실행 순서

서비스 전체를 실행할 때는 아래 순서로 실행한다.

| 순서 | 대상                | 명령어                    | 실행 위치                  |
| ---: | ------------------- | ------------------------- | -------------------------- |
|    1 | Oracle DB           | DB 서버 실행 및 접속 확인 | Oracle DB                  |

## 6. 개발 서버 Proxy 설정

React 개발 서버는 `frontend/src/setupProxy.js`에서 Spring Boot와 Flask 서버로 요청을 분기한다.

| React 호출 경로 | 실제 서버               | 설명            |
| --------------- | ----------------------- | --------------- |
| `/spring`       | `http://localhost:8080` | Spring Boot API |
| `/flask`        | `http://localhost:5000` | Flask AI API    |

예시:

```text
/spring/member/list  ->  http://localhost:8080/member/list
/flask/heart_predict ->  http://localhost:5000/heart_predict
```

## 9. 오류 발생 시 확인 사항

| 증상                 | 확인 사항           | 조치                                                                                    |
| -------------------- | ------------------- | --------------------------------------------------------------------------------------- |
| Java 실행 오류       | JDK 버전            | JDK 17 설치 및 `JAVA_HOME` 확인                                                         |
| Gradle 빌드 실패     | 네트워크, JDK 버전  | 인터넷 연결 및 `java -version` 확인                                                     |
| Oracle 접속 실패     | DB 주소, 계정, 포트 | `210.119.12.109:1521/xe` 접속 가능 여부 확인                                            |
| React 실행 실패      | npm 의존성          | `npm install` 재실행                                                                    |
| npm 명령 인식 오류   | Node.js 설치/Path   | Node.js 재설치 후 새 cmd 창에서 `npm -v` 확인                                           |
| Flask import 오류    | Python 패키지 누락  | `python -m pip install flask flask-cors oracledb joblib numpy pandas scikit-learn` 실행 |
| Flask 모델 로드 오류 | 모델 파일 위치      | `models/heart_model_last.pkl`, `models/model_features.pkl` 확인                         |
| CORS 오류            | Proxy 설정          | `frontend/src/setupProxy.js`의 `/spring`, `/flask` 설정 확인                            |
